# Aiffel_quest_cr

- AIFFEL 과정 중 진행하는 프로젝트와 피어리뷰를 관리하는 Repository입니다.
- 기본적인 프로젝트 구조를 제공하며, 파일이나 폴더를 수정/업로드하여 사용합니다.
- **주의:** Flutter 프로젝트는 반드시 프로젝트 폴더 전체를 업로드합니다.

---

# Data Analysis

이 폴더는 AIFFEL 과정 중 **Data Analysis(데이터 분석)** 파트에서 진행한 퀘스트를 정리하기 위한 공간입니다.  
각 퀘스트 폴더에는 실습용 Jupyter Notebook(.ipynb)과 간단한 설명을 담은 README.md 파일이 포함되어 있습니다.

데이터 분석의 기초 개념부터 실제 데이터를 다루는 과정까지의 실습 결과를 체계적으로 관리하는 것을 목표로 합니다.

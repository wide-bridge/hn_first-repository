# Aiffel_quest_cr

- AIFFEL 과정 중 진행하는 프로젝트와 피어리뷰를 관리하는 Repository입니다.
- 기본적인 프로젝트 구조를 제공하며, 파일이나 폴더를 수정/업로드하여 사용합니다.
- **주의:** Flutter 프로젝트는 반드시 프로젝트 폴더 전체를 업로드합니다.

---

# MLP

이 폴더는 AIFFEL 과정 중 **MLP (Machine Learning Pipeline)** 파트에서 진행한 퀘스트를 정리하기 위한 공간입니다.  
머신러닝 파이프라인의 전체 흐름을 이해하고, 데이터 전처리부터 모델 학습 및 평가까지의 과정을 실습합니다.

효율적인 파이프라인 구성과 재사용 가능한 머신러닝 코드 구조를 학습하는 것을 목표로 합니다.
